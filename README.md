# Advanced Predictive & Prescriptive Procurement Analytics 

An end-to-end data science project utilizing **Python (Pandas, Statsmodels, SciPy)** and **Advanced MS Excel** to optimize supply chain demand forecasting, pricing baselines, and resource allocation across 10,000 procurement transactions.

## Frameworks Implemented
* **Time Series Forecasting (Nada Sanders Framework):** 6-month rolling Holt-Winters Exponential Smoothing model to isolate operational demand trends and annual seasonality cycles.
* **Should-Cost Regression (Kirit Pandit Framework):** Multiple Linear Regression (OLS) modeling to define causal pricing baselines, identify significant cost drivers, and isolate contract price leakage.
* **Prescriptive Optimization (Christian Mandl Framework):** Multi-Objective Linear Programming (MILP) allocation model built to minimize cumulative corporate spend while strictly honoring supplier capacity and risk constraints.

## Repository Contents
* `Final_Procurement_Predictive_Analytics_Report.xlsx`: Integrated executive dashboard containing live financial KPI modules, conditional compliance formatting, and automated charting tools.
* `procurement_analytics_script.py`: Production-grade Python scripts executing time series calculations, OLS diagnostic arrays, and optimization matrix formulations.

## Key Insights
* Successfully projected 2026 enterprise spend vectors over a 3-year historical timeline.
* Identified variance anomalies and targeted pricing leakage for immediate strategic vendor renegotiation.
* Mathematically resolved an optimal allocation matrix for a 25,000-unit fulfillment run under tight capacity constraints.
